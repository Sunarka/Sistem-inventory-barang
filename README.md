## Project Description
This project is a **web-based Inventory Management System** built using **Laravel 12**.  
The system is designed to manage product (barang) data efficiently with full **CRUD functionality** and **REST API integration**.

The application supports:
- Secure authentication (Login & Register)
- Inventory data management
- RESTful API for external access (tested using Postman)

This project was developed as part of an academic assignment focusing on **Laravel MVC architecture and API development**.

---

## Main Features
### Authentication
- Login
- Register
- Logout
- Protected routes using middleware (`auth`)

### Inventory Management (CRUD)
- View product list
- Add new product
- Edit product data
- Delete product data

### REST API
- API endpoints for inventory data
- Supports:
  - **GET** (retrieve data)
  - **POST** (create data)
  - **PUT** (update data)
  - **DELETE** (remove data)
- API tested using **Postman**
- No image upload required for API requests
